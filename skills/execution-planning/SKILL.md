# 执行规划技能

## 技能描述
专业化的任务分解和执行规划技能，基于需求对齐结果生成详细、可执行的执行计划。

## 核心能力
- **任务分解**：将复杂需求分解为1-4小时的可执行任务
- **依赖关系分析**：确定任务间的依赖关系和最优执行顺序
- **验收标准设定**：为每个任务设定明确、可验证的验收条件
- **质量门禁配置**：建立质量检查点和控制机制

## 使用场景
- 需求对齐完成后制定执行计划
- 项目范围调整时重新规划任务
- 执行过程中需要优化执行策略
- 验现有计划的可行性

## 输出产物
生成`EXECUTION_PLAN.json`文件，包含：
- 完整的任务清单和执行顺序
- 每个任务的详细验收标准
- 执行策略和资源配置
- 风险控制和应急预案

## 质量标准
- 任务粒度合理（1-4小时完成）
- 依赖关系清晰准确
- 验收标准具体可验证
- 风险识别覆盖率 ≥ 90%
